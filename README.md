##WebRTC Monitor

Chrome Extension used to monitor WebRTC traffic. Currently only support
Video Traffic.

![](https://raw.githubusercontent.com/plasmashadow/webrtc-monitor/master/shreenshot.png)

##Instruction

Currently Works on sites which uses adapter.js.
